# Título — Notibot References

## Componentes Externos Implementados
- [NestJS Framework](https://nestjs.com/)
- [Baileys WhatsApp Web API](https://github.com/WhiskeySockets/Baileys)
- [Telegraf (Telegram Bot API)](https://telegraf.js.org/)

La estructura sigue convencionalmente NestJS y el controlador principal donde se registran las rutas que puedes ver en openapi es `/api/v1/notifications`.

También tenemos configurado un sistema robusto de testing para ejecutar a través de Jest.
